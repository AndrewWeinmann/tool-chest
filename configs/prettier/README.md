# Prettier

Configuration files for [Prettier](https://prettier.io/), a code formatter.

## `.prettierignore`

Files and directories listed in this file will be ignored by Prettier. See the [Prettier documentation](https://prettier.io/docs/en/ignore) for more details.

## `.prettierrc`

Configuration options for Prettier. See the [Prettier documentation](https://prettier.io/docs/en/options) for available options.
