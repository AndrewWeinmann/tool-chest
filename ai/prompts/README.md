# Prompts

Prompts for AI coding agents.

## Code Review

Do a thorough code review without making any code changes and produce detailed documentation of findings and recommendations.

## Commit History

Create a git commit history for uncommitted changes.

## Commit Staged

Create a git commit for the staged changes only.

## Rewrite History

Rewrite the git commit history for a branch to produce a clear, coherent, and building story of the development.
